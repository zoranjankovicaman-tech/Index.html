Aman — Podešavanje Firebase baze

🔧 Jednokratno podešavanje — Firebase baza podataka
1
Otvorite console.firebase.google.com i prijavite se Google nalogom
2
Kliknite "Add project" → naziv projekta: knjiga-rokova-aman → Continue → Continue → Create project
3
U lijevom meniju kliknite "Realtime Database" → "Create Database" → Izaberite Europe (belgium) → "Start in test mode" → Enable
4
Kliknite na zupčanik (⚙️) → Project settings → tab "Your apps" → ikona </> (web) → App nickname: knjiga-rokova → Register app
5
Vidjet ćete kod koji počinje sa const firebaseConfig = { ... }. Kopirajte sve unutar vitičastih zagrada { ... } uključujući i same zagrade i nalijepite dolje.
6
U Firebase konzoli → Realtime Database → Rules → zamijenite sadržaj sa:
{ "rules": { ".read": true, ".write": true } }
Kliknite Publish
📋 Nalijepite Firebase konfiguraciju

📧 Automatski Email Izvještaji — Google Apps Script

Ovaj skript šalje automatski email svakoj prodavnici svaki dan u 6:00 sa listom artikala za sutrašnji dan.

Koraci za podešavanje:

  1. Otvorite script.google.com → New project
  2. Zamijenite cijeli sadržaj sa kodom ispod
  3. Promijenite FIREBASE_URL u svoju Firebase URL adresu
  4. Kliknite Save → Run → createTrigger (odobrite pristup)
  5. Skript će se automatski pokretati svaki dan u 6:00
